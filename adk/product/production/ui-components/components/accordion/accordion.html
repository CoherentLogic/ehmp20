Accordion

Description: An accordion allows the user to view lengthy content quickly by browsing through various topics. Once the user has identified the topic, selecting the topic title will display content that fall below the title. The user can only view one topic at a time. As the user selects another title, the previous title will close creating an accordion effect.

508 Guidelines: In order to make accordions 508 complaint for a keyboard user, adding href="#" to the heading will allow the user to tab to the link and select Enter on the keyboard to open and close the accordion.

TIP: The first accordion will automatically be expanded. To stop this functionality and close all accordions remove the class .in; invertly, if every accordion needs to be open add the class .in.
